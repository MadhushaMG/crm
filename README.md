# Multi-Tenant CRM System

A production-ready CRM built with **Django REST Framework** and **React**.

## Features
- **Multi-tenancy:** Strict data isolation between organizations.
- **Role-Based Access (RBAC):** Admin, Manager, and Staff roles.
- **Cloud Storage:** Image uploads integrated with **AWS S3**.
- **Audit Logs:** Automated activity tracking for all CRUD actions.
- **Security:** JWT Authentication and Custom Permission classes.

## Tech Stack
- **Backend:** Python, Django, DRF, PostgreSQL.
- **Frontend:** React, Zustand, Axios.
- **Infrastructure:** AWS S3, Git.
